## Дискретное преобразование Фурье
Вычисление дискретнонр преобразования Фурье (ДПФ) функции _**f(x) = x(1 −
x)ln(1 − x).**_
1. Реализация ДПФ для числа узлов N = 10,50,100,1000;
2. Вычисление обратного преобразования Фурье для полученных коэффициентов, выведение
погрешности и построение графиков приближения;
